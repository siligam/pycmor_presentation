# PyCMOR Presentation

A presentation about PyCMOR - Empowering scientists in CMORization.

## Development

```bash
npm install
npm run dev
```

## Deployment

This presentation is automatically deployed to GitHub Pages
when changes are pushed to the main branch.

The deployment workflow:

1. Builds the Slidev presentation
2. Deploys to GitHub Pages at `https://siligam.github.io/pycmor_presentation/`

## Local Build

```bash
npm run build
```

The built files will be in the `dist/` directory.
